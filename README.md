## API Rest Node Express

Endpoints:
| Rota | Método | Status Retorno |
| :--- | :--- | :--- |
| `/` | GET | 200 |
| `/api/usuarios` | GET | 200 |
| `/api/usuarios/:id` | GET | 200 ou 404 |
| `/api/usuarios` | POST | 201 ou 400 |
| `/api/usuarios/:id` | PUT | 204 ou 404 |
| `/api/usuarios/:id` | DELETE | 204 ou 404 |
